# Microsoft Sentinel Analytic Rules

This repository contains a curated collection of **Microsoft Sentinel analytic rules** designed to help security teams detect, investigate, and respond to suspicious or malicious activity across their environments.

The focus of this repo is on **practical, high-signal detections** that align with real-world threat scenarios, modern attack techniques, and operational SOC workflows.
